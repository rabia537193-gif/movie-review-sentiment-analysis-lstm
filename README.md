Movie Review Sentiment Analysis Using LSTM
Overview

This project implements a Deep Learning-based Sentiment Analysis model using Long Short-Term Memory (LSTM) networks. The model is trained on the IMDB Movie Reviews dataset to classify reviews as either positive or negative. It demonstrates the use of Natural Language Processing (NLP) techniques such as text preprocessing, sequence padding, word embeddings, and recurrent neural networks.

Features
Binary sentiment classification (Positive/Negative)
LSTM-based deep learning architecture
IMDB movie review dataset
Text preprocessing and tokenization
Model training and evaluation using TensorFlow/Keras
Technologies Used
Python
TensorFlow / Keras
NumPy
IMDB Dataset
Model Architecture
Embedding Layer
LSTM Layer
Dense Output Layer (Sigmoid Activation)
Results

The model achieves high accuracy in classifying movie reviews and effectively captures sentiment from textual data.

Learning Outcomes
Understanding of NLP workflows
Implementation of LSTM networks
Text classification using Deep Learning
Model evaluation and performance analysis
